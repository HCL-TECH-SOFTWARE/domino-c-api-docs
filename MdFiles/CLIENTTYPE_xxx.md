**Initial Release 4.0**



**Symbolic Value : Address Book;
Constants**



**CLIENTTYPE\_xxx** **-** Constants
for ClientType field in Address Book.


**----------------------------------------------------------------------------------------------------------**



**#include <names.h>**


 **Symbolic Values :**      CLIENTTYPE\_FULL             -  Full client (not restricted
as with Notes Express or Desktop clients).  

  

      CLIENTTYPE\_LIMITED        -  Notes Express client.  

  

      CLIENTTYPE\_DESKTOP     -  Desktop client.  

  




**Description :**



Possible
numerical values for the "ClientType" (FIELD\_CLIENTTYPE) item.  This
value is converted to text before being stored in the Address book
