**Initial Release 4.0**



**Symbolic Value : Compound Text**



**ACTIONFOLDER\_FLAG\_xxx** **-** Option flags
for CDACTIONFOLDER record.


**----------------------------------------------------------------------------------------------------------**



**#include <queryods.h>**


 **Symbolic Values :**      ACTIONFOLDER\_FLAG\_NEWFOLDER        -  Create new folder.  

  

      ACTIONFOLDER\_FLAG\_PRIVATEFOLDER              -  Folder is private.  

  




**Description :**



These option
flags may be specified in the dwFlags field of the CDACTIONFOLDER record.
