**Symbolic Value : Rich Text**



**CGM\_MAPMODE\_xxx** **-** Mapping mode
for CGM metafile.


**----------------------------------------------------------------------------------------------------------**



**#include <editods.h>**


 **Symbolic Values :**      CGM\_MAPMODE\_ABSTRACT         -  Virtual coordinate system
(default).  

  

      CGM\_MAPMODE\_METRIC   -  Currently unsupported.  

  




**Description :**



The CGM
mapping mode is stored in the mm field of the CDCGMMETA record.
