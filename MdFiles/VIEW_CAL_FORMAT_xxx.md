**Initial Release 4.5**



**Symbolic Value : Views**



**VIEW\_CAL\_FORMAT\_xxx** **-** VIEW\_CALENDAR\_FORMAT
- Formats


**----------------------------------------------------------------------------------------------------------**



**#include <viewfmt.h>**


 **Symbolic Values :**      VIEW\_CAL\_FORMAT\_TWO\_DAY     -  Two days format.  

  

      VIEW\_CAL\_FORMAT\_ONE\_WEEK   -  One week format.  

  

      VIEW\_CAL\_FORMAT\_TWO\_WEEKS            -  Two weeks format.  

  

      VIEW\_CAL\_FORMAT\_ONE\_MONTH            -  One month format.  

  

      VIEW\_CAL\_FORMAT\_ONE\_YEAR   -  One year format.  

  

      VIEW\_CAL\_FORMAT\_ONE\_DAY      -  One day format.  

  

      VIEW\_CAL\_FORMAT\_WORK\_WEEK           -  Custom work week format.  

  

      VIEW\_CAL\_FORMAT\_ALL    -  Bit mask for VIEW\_CAL\_FORMAT\_xxx values.  

  




**Description :**



These
symbols specify the possible values for the Formats field in a
VIEW\_CALENDAR\_FORMAT structure.
