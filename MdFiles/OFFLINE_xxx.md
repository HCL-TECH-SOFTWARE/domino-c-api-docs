**Initial Release 5.0**



**Symbolic Value : Backup**



**OFFLINE\_xxx** **-** Recovery
options.


**----------------------------------------------------------------------------------------------------------**



**#include <nsfdb.h>**


 **Symbolic Values :**      OFFLINE\_NODELETE          -  Don't delete database file.  

  

      OFFLINE\_KEEP\_ACCESS\_ENTITIES           -  Don't delete DB2 access
entities.  

  




**Description :**



This option
is used when the database is not to be automatically deleted when taken
offline.
