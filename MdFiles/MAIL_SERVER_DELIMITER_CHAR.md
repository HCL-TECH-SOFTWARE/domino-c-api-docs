**Symbolic Value : Mail**



**MAIL\_SERVER\_DELIMITER\_CHAR** **-** Server name
delimiter character.


**----------------------------------------------------------------------------------------------------------**



**#include <mail.h>**



**Description :**



This is the
delimiter used to specify a server within a domain specifier (for example, xxx
@ LOTUS:QASERVER, where LOTUS is the name of the domain and QASERVER is the
name of the server).
