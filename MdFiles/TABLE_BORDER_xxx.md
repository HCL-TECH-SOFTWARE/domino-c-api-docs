**Symbolic Value : Rich Text**



**TABLE\_BORDER\_xxx** **-** Table cell
border types.


**----------------------------------------------------------------------------------------------------------**



**#include <editods.h>**


 **Symbolic Values :**      TABLE\_BORDER\_NONE      -  No border.  

  

      TABLE\_BORDER\_SINGLE   -  Single border.  

  

      TABLE\_BORDER\_DOUBLE              -  Double border.  

  




**Description :**



The border
types are stored in the Borders field of the CDTABLECELL record.
