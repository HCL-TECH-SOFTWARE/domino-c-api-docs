**Initial Release 5.0**



**Data Type : Standard**



**OLE\_BOOL** **-** Boolean
value used by OLE.


**----------------------------------------------------------------------------------------------------------**



**#include
<global.h>**



**Definition :**



#ifdef MAC


typedef unsigned long
OLE\_BOOL  

#else  

typedef BOOL          OLE\_BOOL;  

#endif


 


**Description :**



Boolean
value used by OLE.  This value may be set to either TRUE or FALSE.  OLE uses a
different datatype for a boolean value than BOOL, the datatype that Domino and
Notes uses for a boolean value.
