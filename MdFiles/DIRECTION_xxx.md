**Initial Release 4.5**



**Symbolic Value : Replica Info;
Replication**



**DIRECTION\_xxx** **-** Direction
member of the REPLHIST\_SUMMARY structure.


**----------------------------------------------------------------------------------------------------------**



**#include <nsfdb.h>**


 **Symbolic Values :**      DIRECTION\_NEVER            -  Never  

  

      DIRECTION\_SEND   -  Send  

  

      DIRECTION\_RECEIVE         -  Receive  

  




**Description :**



These values
describe the Direction member of the REPLHIST\_SUMMARY structure (the direction
of the replication in the replication history).
