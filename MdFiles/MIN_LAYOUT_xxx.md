**Initial Release 4.0**



**Symbolic Value : Limits; Rich Text**



**MIN\_LAYOUT\_xxx** **-** Minimum
values for certain attributes of a layout region on a form.


**----------------------------------------------------------------------------------------------------------**



**#include <editdflt.h>**


 **Symbolic Values :**      MIN\_LAYOUT\_WIDTH          -  (ONEINCH / 4)  

  

      MIN\_LAYOUT\_HEIGHT        -  (ONEINCH / 4)  

  




**Description :**



Minimum
values for the height and width of a layout region on a form.  These symbols
may be used for the wWidth and wHeight members of the CDLAYOUT structure.
