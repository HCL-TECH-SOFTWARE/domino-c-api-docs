**Initial Release 5.0**



**Symbolic Value : Domino Upgrade
Services**



**DUS\_GROUP\_TYPE\_xxx** **-** Used to
write to the ADMINP\_GROUP\_TYPE field in the group note.


**----------------------------------------------------------------------------------------------------------**



**#include <dus.h>**


 **Symbolic Values :**      DUS\_GROUP\_TYPE\_MULTI\_PURPOSE      -  Group is multi-purpose.  

  

      DUS\_GROUP\_TYPE\_ACL\_ONLY     -  Group is access control list only.  

  

      DUS\_GROUP\_TYPE\_MAIL\_ONLY    -  Group is mail only.  

  

      DUS\_GROUP\_TYPE\_DENY\_LIST\_ONLY      -  Group is deny list only.
