**NCURFMT\_xxx** **-** CDEXT2FIELD
- CurrencyType and CurrencyFlags data member values.


**----------------------------------------------------------------------------------------------------------**



**#include <misc.h>**


 **Symbolic Values :**      NCURFMT\_SYMFOLLOWS   -  CurrencyFlags - The currency symbol
follows the value.  

  

      NCURFMT\_USESPACES     -  CurrencyFlags - Insert space between symbol and
value.  

  

      NCURFMT\_ISOSYMUSED   -  CurrencyFlags - Using 3 letter ISO for currency
symbol.  

  

      NCURFMT\_COMMON          -  CurrencyType - Currency selection value
common.  

  

      NCURFMT\_CUSTOM           -  CurrencyType - Currency selection value
custom.  

  




**Description :**



The 
currency format for the CurrencyType and CurrencyFlags members of CDEXT2FIELD.
