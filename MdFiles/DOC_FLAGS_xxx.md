**Symbolic Value : OLE Data Object**



**DOC\_FLAGS\_xxx** **-** State
information for an OLE object.


**----------------------------------------------------------------------------------------------------------**



**#include <olenotes.h>**


 **Symbolic Values :**      DOC\_FLAGS\_EDITFORM     -  A form is being edited.  

  

      DOC\_FLAGS\_COMPOSE\_NEW        -  A new document is being composed, but it
has not yet been saved.  

  

      DOC\_FLAGS\_WIN32            -  Windows 32-bit version of Notes.  

  

      DOC\_FLAGS\_MAC   -  Macintosh version of Notes.  

  




**Description :**



This symbol
is used to define some state information for documents containing OLE objects
