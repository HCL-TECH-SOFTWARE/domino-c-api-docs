**Initial Release 6**



**Symbolic Value : Constants; Rich
Text**



**ACTIONBAR\_BUTTON\_WIDTH\_xxx** **-** Values for
BtnWidthStyle member of CDACTIONBAREXT.


**----------------------------------------------------------------------------------------------------------**



**#include <actods.h>**


 **Symbolic Values :**      ACTIONBAR\_BUTTON\_WIDTH\_DEFAULT   -  Action button width is
calculated based on text length and image width.  

  

      ACTIONBAR\_BUTTON\_WIDTH\_BACKGROUND      -  Action button width is at least
button background image width or wider if needed to fit text and image.  

  

      ACTIONBAR\_BUTTON\_WIDTH\_ABSOLUTE            -  Action button width is
fixed. If this is set, the wBtnWidthAbsolute member of CDACTIONBAREXT reflects
the fixed width.  

  




**Description :**



These values
for the BtnWidthStyle member of CDACTIONBAREXT describe action button width
styles.
