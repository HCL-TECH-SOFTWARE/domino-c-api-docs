**Symbolic Value : Rich Text**



**LAYOUT\_GRAPHIC\_FLAG\_xxx** **-** CDLAYOUTGRAPHIC
flags


**----------------------------------------------------------------------------------------------------------**



**#include <editods.h>**


 **Symbolic Values :**      LAYOUT\_GRAPHIC\_FLAG\_BUTTON           -  The graphical
element is also used as a button.  

  




**Description :**



These flags
are set in the "Flags" field of a CDLAYOUTGRAPHIC record, and control
options for the graphical element in the layout region.
