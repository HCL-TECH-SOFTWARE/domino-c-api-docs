**Symbolic Value : Views**



**VIEW\_CALENDAR\_FORMAT2\_SIGNATURE** **-** Value for
the Signature member of the VIEW\_CALENDAR\_FORMAT2 structure.


**----------------------------------------------------------------------------------------------------------**



**#include <viewfmt.h>**


 **Symbolic Values :**      VIEW\_CALENDAR\_FORMAT2\_SIGNATURE            -  Signature of a
secondary calendar view format.  

  




**Description :**



The VIEW\_CALENDAR\_FORMAT2
record begins with a WORD value for the Signature of the record.  This symbol
specifies the signature of the VIEW\_CALENDAR\_FORMAT2 record.
