**Initial Release 4.0**



**Symbolic Value : Search**



**PERCENTILE\_COUNT** **-** Number of
entries in percentile table


**----------------------------------------------------------------------------------------------------------**



**#include <nif.h>**



**Description :**



Keys in a
COLLECTIONDATA structure are divided into percentiles - divisions corresponding
to one-tenth of the total range of keys - and a table of the keys marking the
divisions is returned with that structure.  This constant gives the number of
entries in that table.
