**#include <ldap.h>**


 **Symbolic Values :**      LDAP\_SASL\_SIMPLE           -  Use simple authentication.
Much like using the simplified routines ldap\_simple\_bind or ldap\_simple\_bind\_s.  

  

      LDAP\_SASL\_EXTERNAL      -  Perform SASL bind with the EXTERNAL mechanism.
