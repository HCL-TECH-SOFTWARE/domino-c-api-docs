**Symbolic Value : MIME**



**MIME\_PART\_xxx** **-** MIME\_PART -
dwFlags


**----------------------------------------------------------------------------------------------------------**



**#include <mimeods.h>**


 **Symbolic Values :**      MIME\_PART\_HAS\_BOUNDARY       -  Mime part has boundary.  

  

      MIME\_PART\_HAS\_HEADERS          -  Mime part has headers.  

  

      MIME\_PART\_BODY\_IN\_DBOBJECT            -  Mime part has body in database
object.  

  

      MIME\_PART\_SHARED\_DBOBJECT             -  Mime part has shared database
object. Used only with MIME\_PART\_BODY\_IN\_DBOBJECT.  

  

      MIME\_PART\_SKIP\_FOR\_CONVERSION      -  Skip for conversion.  

  




**Description :**



Possible
values for the dwFlags member of the MIME\_PART structure.
