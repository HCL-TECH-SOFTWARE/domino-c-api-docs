**Symbolic Value : Time**



**TTFMT\_xxx** **-** TFMT -
Time.  Specifies the Time format for character time/date strings.


**----------------------------------------------------------------------------------------------------------**



**#include <misc.h>**


 **Symbolic Values :**      TTFMT\_FULL           -  Time format is: hour, minute, and
second  

  

      TTFMT\_PARTIAL     -  Time format is: hour and minute  

  

      TTFMT\_HOUR         -  Time format is: hour  

  




**Description :**



These
definitions specify the format for character time/date strings. Use these
definitions when you set up the TFMT structure that controls the time/date
format.
