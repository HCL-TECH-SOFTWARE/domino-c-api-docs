**Symbolic Value : Time**



**TIMEDATE\_xxx** **-** Set of
TIMEDATE value constants.


**----------------------------------------------------------------------------------------------------------**



**#include <misc.h>**


 **Symbolic Values :**      TIMEDATE\_MINIMUM          -  Clear the TIMEDATE value.  

  

      TIMEDATE\_MAXIMUM         -  Set the TIMEDATE value to the maximum value.  

  

      TIMEDATE\_WILDCARD       -  Set the TIMEDATE value to ANYDAY/ALLDAY  

  




**Description :**



These
definitions specify the set of TIMEDATE value constants.
