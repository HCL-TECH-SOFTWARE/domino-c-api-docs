**Symbolic Value : Views**



**CALENDAR\_TYPE\_xxx** **-** Types of
calendars.


**----------------------------------------------------------------------------------------------------------**



**#include <viewfmt.h>**


 **Symbolic Values :**      CALENDAR\_TYPE\_DAY       -  Day style calendar.  

  

      CALENDAR\_TYPE\_WEEK    -  Week style calendar.  

  

      CALENDAR\_TYPE\_MONTH             -  Month style calendar  

  




**Description :**



These
symbols describe the type of calendar.  The type of calendar is specified in
the lower four bits of the BYTE ViewStyle member of the VIEW\_FORMAT\_HEADER
structure.
