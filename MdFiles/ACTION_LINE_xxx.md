**Initial Release 4.6.1**



**Symbolic Value : Form**



**ACTION\_LINE\_xxx** **-** CDACTIONBAR
Line Styles.


**----------------------------------------------------------------------------------------------------------**



**#include <actprop.h>**


 **Symbolic Values :**      ACTION\_LINE\_SINGLE        -  CDACTIONBAR dividing line
height of 1.  

  

      ACTION\_LINE\_DOUBLE       -  CDACTIONBAR dividing line height of 2.  

  

      ACTION\_LINE\_TRIPLE        -  CDACTIONBAR dividing line height of 3.  

  

      ACTION\_LINE\_TWO            -  CDACTIONBAR dividing line height of 2
separate lines.  

  




**Description :**



The
CDACTIONBAR structure contains a LineStyle member which may be set to one of
the above.  If the BorderStyle member is set to ACTION\_BORDER\_NONE  LineStyle
is ignored.
