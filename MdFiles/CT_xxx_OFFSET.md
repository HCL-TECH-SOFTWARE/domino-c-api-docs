**Symbolic Value : Rich Text**



**CT\_xxx\_OFFSET** **-** Offsets of
red, green, and blue color values


**----------------------------------------------------------------------------------------------------------**



**#include <editods.h>**


 **Symbolic Values :**      CT\_RED\_OFFSET    -  Offset to the Red color byte (byte 0)  

  

      CT\_GREEN\_OFFSET           -  Offset to the Green color byte (byte 1)  

  

      CT\_BLUE\_OFFSET   -  Offset to the Blue color byte (byte 2)  

  




**Description :**



These
constants are offsets to the Red, Green, and Blue color values in a
CDCOLORTABLE color entry.  These color values are used in color tables and
CDPATTERNTABLE records.
