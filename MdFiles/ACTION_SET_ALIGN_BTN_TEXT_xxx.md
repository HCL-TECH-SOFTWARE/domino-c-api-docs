**Initial Release 6**



**Symbolic Value : Constants; Rich
Text**



**ACTION\_SET\_ALIGN\_BTN\_TEXT\_xxx** **-** Values for
BtnTextJustify member of CDACTIONBAREXT.


**----------------------------------------------------------------------------------------------------------**



**#include <actprop.h>**


 **Symbolic Values :**      ACTION\_SET\_ALIGN\_BTN\_TEXT\_LEFT      -  Text in action button
is left justified within button.  

  

      ACTION\_SET\_ALIGN\_BTN\_TEXT\_CENTER            -  Text in action button is
centered within button.  

  

      ACTION\_SET\_ALIGN\_BTN\_TEXT\_RIGHT    -  Text in action button is right
justified within button.
