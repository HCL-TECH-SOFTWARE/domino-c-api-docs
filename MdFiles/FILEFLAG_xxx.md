**Symbolic Value : File Attachment**



**FILEFLAG\_xxx** **-** Miscellanous
FILEOBJECT flags.


**----------------------------------------------------------------------------------------------------------**



**#include <nsfdata.h>**


 **Symbolic Values :**      FILEFLAG\_SIGN      -  File object has signature appended.  

  

      FILEFLAG\_INDOC    -  File is represented by an editor run in the
document.  

  

      FILEFLAG\_MIME      -  File object has mime data appended.  

  

      FILEFLAG\_AUTOCOMPRESSED      -  File is a folder automaticly compressed
by Notes.  

  




**Description :**



These values
may be used in the Flags member of the FILEOBJECT structure.
