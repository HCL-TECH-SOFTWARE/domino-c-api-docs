**Symbolic Value : User Registration**

**#include <misc.h>**


 **Symbolic Values :**      SEC\_pwddigest\_V1   -  Original, insecure password digest  

  

      SEC\_pwddigest\_V2   -  More secure, compatible with R6+  

  

      SEC\_pwddigest\_V3   -  Even more secure compatible with 8.01+
