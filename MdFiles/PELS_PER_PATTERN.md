**Symbolic Value : Rich Text**



**PELS\_PER\_PATTERN** **-** Number of
Picture ELementS in a pattern table entry


**----------------------------------------------------------------------------------------------------------**



**#include <editods.h>**



**Description :**



A pattern
table is a fixed-size color table used for patterns by CDBITMAPHEADER (patterns
are used to compress the bitmap). An entry in the pattern table is 8
(PELS\_PER\_PATTERN) packed colors (3 bytes per color).
