**Initial Release 4.1**



**Symbolic Value : Billing**



**BILL\_AGENT\_xxx** **-** AGENTREC -
Flags


**----------------------------------------------------------------------------------------------------------**



**#include <billing.h>**


 **Symbolic Values :**      BILL\_AGENT\_WARNINGS    -  Warnings agent billing flag  

  

      BILL\_AGENT\_MAIL   -  Mail agent billing flag  

  

      BILL\_AGENT\_PUBLIC          -  Public agent billing flag  

  




**Description :**



These
constants identify the type of agent operation that occurred and generated an
agent billing flag.  They are the possible values for the Flags member of the
AGENTREC structure.
