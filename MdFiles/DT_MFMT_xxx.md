**DT\_MFMT\_xxx** **-** CDEXT2FIELD
- DTMonthFmt


**----------------------------------------------------------------------------------------------------------**



**#include <misc.h>**


 **Symbolic Values :**      DT\_MFMT\_M           -  Display month value without leading
zero.  

  

      DT\_MFMT\_MM        -  Display month value with leading zero.  

  

      DT\_MFMT\_MMM      -  Display month in abbreviated format. For example:
Jan.  

  

      DT\_MFMT\_MMMM   -  Display month fully spelled out. For example: January.  

  




**Description :**



The
"Month" format of the "Display Date".
