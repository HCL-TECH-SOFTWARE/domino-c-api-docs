##### Symbolic Value : Note
##### RRV_xxx - Note ID size and deletion mask.
---
##### #include <nsfdata.h>
**Description :**
The size of a Note ID to be used in a call to IDTableCreate().  The mask that 
is OR'd with all Note IDs (NOTEIDs) returned by NSFDbGetModifiedNoteTable().
