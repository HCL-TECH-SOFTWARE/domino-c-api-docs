##### Symbolic Value : DDE
##### DDEFORMAT_xxx - DDE clipboard format.
---
```
#include <editods.h>
```
**Description :**

These values specify the DDE clipboard format with which data should be 
rendered.  They are used in the CDDDEBEGIN structure and in the CDOLEBEGIN 
structure.
