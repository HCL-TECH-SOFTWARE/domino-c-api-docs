##### Symbolic Value : Limits
##### MINSIZEFORDOCTRUNCATION - Truncate document whose size is > 40K
---
```
#include <misc.h>
```

**Symbolic Values :**



**Description :**

This symbol defines If Document modified TIMEDATE is greater than LastMMRTruncationCutoff and less than Cutoff TIMEDATE (i.e. 24 hrs after LastMMRTruncationCutof) and Document size is &gt; 40K then only consider document for truncation.
