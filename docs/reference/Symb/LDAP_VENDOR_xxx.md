##### Symbolic Value : LDAP
##### LDAP_VENDOR_xxx - Compile time vendor information.
---
```
#include <ldap.h>
```

**Symbolic Values :**

	LDAP_VENDOR_NAME	  -  "IBM Lotus Software"

	LDAP_VENDOR_VERSION	  -  100 (Version 1.0)


**Description :**

This vendor information is used at compile time.
