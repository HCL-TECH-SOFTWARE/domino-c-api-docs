##### Symbolic Value : Alarms
##### ALARM_DATA_FLAG_xxx - Alarm flags.
---
```
#include <alarm.h>
```

**Symbolic Values :**

	ALARM_DATA_FLAG_DISPLAY_ALARM	  -  Display the alarm

	ALARM_DATA_FLAG_PLAY_TUNE	  -  Play the tune

	ALARM_DATA_FLAG_SEND_MAIL	  -  Send mail


**Description :**

Used for the Alarm API.
