##### Symbolic Value : Constants
##### NULLHCOLLECTION - Null collection handle.
---
```
#include <nif.h>
```
**Description :**

A NULLHCOLLECTION is an empty handle for functions having an HCOLLECTION 
parameter.

It is defined as (HCOLLECTION) 0.
