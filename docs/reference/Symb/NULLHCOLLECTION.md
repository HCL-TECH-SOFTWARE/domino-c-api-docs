##### Symbolic Value : Constants
##### NULLHCOLLECTION - Null collection handle.
---
```
#include <nif.h>
```

**Symbolic Values :**



**Description :**

A NULLHCOLLECTION is an empty handle for functions having an HCOLLECTION parameter.
<ul><br>
<br>
It is defined as (HCOLLECTION) 0.</ul>
