##### Symbolic Value : LDAP
##### LDAP_OPT_xxx - LDAP options.
---
```
#include <ldap.h>
```
**Description :**

LDAP options that can be accessed or set using the functions ldap_get_option 
and ldap_set_option.  Type may differ dependant on whether option is to be 
accessed or set.
