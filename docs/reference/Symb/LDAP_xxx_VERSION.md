##### Symbolic Value : LDAP
##### LDAP_xxx_VERSION - Compile time version information.
---
```
#include <ldap.h>
```

**Symbolic Values :**

	LDAP_SPEC_VERSION	  -  Specification version.

	LDAP_API_VERSION	  -  API version.


**Description :**

This version information is used at compile time.
