##### Symbolic Value : Basic Encoding Rules
##### NULLBER - An empty pointer to a BerElement
---
```
#include <lber.h>
```

**Symbolic Values :**

	NULLBER	  -  An empty pointer to a BerElement.


**Description :**

NULLBER is an empty pointer to a BerElement, defined as: ((BerElement *) 0).
