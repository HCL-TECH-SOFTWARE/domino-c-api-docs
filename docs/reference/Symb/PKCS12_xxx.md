##### Symbolic Value : IDs
##### PKCS12_xxx - Merging keys and certifications into ID file options.
---
```
#include <pkcs12.h>
```

**Symbolic Values :**

	PKCS12_EXCLUDE_PRIVATEKEYS	  -  Do not process keys.


**Description :**

Options for merging keys and certifications into an ID file.
