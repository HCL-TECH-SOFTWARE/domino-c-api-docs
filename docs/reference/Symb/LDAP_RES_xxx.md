##### Symbolic Value : LDAP
##### LDAP_RES_xxx - Possible result types a server can return.
---
```
#include <ldap.h>
```
**Description :**

Possible result types a server can return.  Upon successful completion, 
ldap_result returns the type of the first result. This will be one of the 
following constants.
