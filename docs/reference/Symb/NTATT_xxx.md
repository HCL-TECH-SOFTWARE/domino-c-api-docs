##### Symbolic Value : File Attachment
##### NTATT_xxx - File attachment - file types.
---
```
#include <nsfdata.h>
```
**Description :**

These values are optional values for the encoding_type parameter of 
NSFNoteAttachFile.  It specifies the file type of a file attachment.
