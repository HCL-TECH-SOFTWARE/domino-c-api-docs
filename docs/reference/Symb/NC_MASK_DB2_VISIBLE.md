##### Symbolic Value : DB2
##### NC_MASK_DB2_VISIBLE -  All note classes visible to DB2.
---
```
#include <nsfnote.h>
```
**Description :**

 All note classes up to NOTE_CLASS_REPLFORMULA visible to DB2.  See the 
definitions for NOTE_CLASS_xxx in nsfnote.h.
