##### Symbolic Value : Agents
##### QUERYFORMULA_FLAG_xxx - Option flags for Formula query.
---
```
#include <queryods.h>
```

**Symbolic Values :**

	QUERYFORMULA_FLAG_PLAINTEXT	  -  Show formula as plain text.


**Description :**

These flags are stored in the dwFlags of the CDQUERYFORMULA record.
