##### Symbolic Value : Agents
##### QUERY_FLAG_xxx - Option flags for agent query.
---
```
#include <queryods.h>
```

**Symbolic Values :**

	QUERY_FLAG_INCLUDEDESCENDANTS	  -  Include all children.


**Description :**

These flags are stored in the dwFlags field of the CDQUERYHEADER record.
