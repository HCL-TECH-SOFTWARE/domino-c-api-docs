##### Symbolic Value : LDAP
##### LDAP_NO_LIMIT - LDAP value
---
```
#include <ldap.h>
```

**Symbolic Values :**

	LDAP_NO_LIMIT	  -  There is no limit.


**Description :**

Possible LDAP value for LDAP_SIZE_LIMIT and LDAP_TIMELIMIT.
