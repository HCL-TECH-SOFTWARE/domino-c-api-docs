##### Symbolic Value : MIME
##### SIGN_NOTES_IF_MIME_PRESENT - Flag for NSFNoteSignExt3
---
```
#include <nsfnote.h>
```
**Description :**

 If the note has MIME parts, this Flag will allow it to be Notes signed.
