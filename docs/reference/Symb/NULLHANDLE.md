##### Symbolic Value : Constants
##### NULLHANDLE - A null handle.
---
```
#include <global.h>
```

**Symbolic Values :**



**Description :**

A NULLHANDLE is a universal empty handle for the Notes API.  It may be used as an empty DBHANDLE, NOTEHANDLE, FORMULAHANDLE, or SEARCHHANDLE.

<br>
