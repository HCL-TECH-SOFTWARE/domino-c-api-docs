##### Symbolic Value : Access Control List
##### ACL_PRIVNAMEMAX - Maximum length of a privilege or role name.
---
```
#include <acl.h>
```
**Description :**

Maximum length of a privilege or role name (not including the paretheses or 
square bracket delimiters).
