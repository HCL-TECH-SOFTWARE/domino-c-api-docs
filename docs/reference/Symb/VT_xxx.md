##### Symbolic Value : Statistics Reporting
##### VT_xxx - Value types of statistics.
---
```
#include <stats.h>
```

**Symbolic Values :**

	VT_LONG	  -  Statistic is of type LONG.

	VT_TEXT	  -  Statistic is of type TEXT.

	VT_TIMEDATE	  -  Statistic is of type TIMEDATE.

	VT_NUMBER	  -  Statistic is of type NUMBER.


**Description :**

These values specify the value types of a statistic.
