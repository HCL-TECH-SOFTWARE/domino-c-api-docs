##### Symbolic Value : Note
##### NOTE_LOCK_xxx - Flags to control how a note is locked.
---
```
#include <nsfnote.h>
```

**Symbolic Values :**

	NOTE_LOCK_STATUS	  -  Return status of lock.

	NOTE_LOCK_HARD	  -  Take out a hard note lock.

	NOTE_LOCK_PROVISIONAL	  -  Take out a provisional hard note lock.
