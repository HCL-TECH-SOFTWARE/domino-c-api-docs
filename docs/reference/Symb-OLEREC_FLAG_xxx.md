##### Symbolic Value : OLE
##### OLEREC_FLAG_xxx - OLE object definition flags.
---
##### #include <editods.h>
**Description :**
These flags are used to define the type of OLE object a note contains.  These 
flags are used by the Flags member of the CDOLEBEGIN data structure.
