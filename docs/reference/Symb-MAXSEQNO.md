##### Symbolic Value : IDs
##### MAXSEQNO - ORIGINATORID - Sequence
---
##### #include <nsfdata.h>
**Description :**
MAXSEQNO is the maximum sequence number for a note.  Use this symbol to check 
if the Sequence member of the ORIGINATORID structure is equal to this.
