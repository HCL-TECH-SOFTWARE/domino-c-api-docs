##### Data Type : Views
##### COLLECTIONSTATS - Returned by NIFReadEntries.
---
##### #include <nif.h>
**Description :**
If requested, this structure is returned by NIFReadEntries at the front of the 
returned information buffer.  The structure describes statistics about the 
overall collection.
