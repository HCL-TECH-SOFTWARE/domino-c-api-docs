##### Symbolic Value : Constants
##### NULLHCERTIFIER - Null certifier handle.
---
##### #include <kfm.h>
**Description :**
A NULLHCERTIFIER is an empty handle for functions having an HCERTIFIER 
parameter.

It is defined as (HCERTIFIER) 0.
