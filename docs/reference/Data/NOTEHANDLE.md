##### Data Type : Handles
##### NOTEHANDLE - Note handle.
---
```
#include <nsfdata.h>
```

**Definition :**

#define NOTEHANDLE DHANDLE

**Description :**

A NOTEHANDLE is the method by which an API program refers to an individual note, once the note is opened.
