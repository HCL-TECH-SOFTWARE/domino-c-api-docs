##### Data Type : Handles
##### FORMULAHANDLE - Formula handle.
---
```
#include <nsfdata.h>
```

**Definition :**
```
#define FORMULAHANDLE HANDLE
```

**Description :**

A FORMULAHANDLE is the method by which an API program refers to a compiled formula.
