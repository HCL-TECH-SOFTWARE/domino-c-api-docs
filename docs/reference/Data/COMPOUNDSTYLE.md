##### Data Type : Rich Text
##### COMPOUNDSTYLE - Defines a format for CompoundText
---
```
#include <easycd.h>
```
**Description :**

This structure specifies paragraph attributes for a style.  It is used  for 
CompoundText paragraphs and text.  It is similar to the CDPABDEFINITION 
structure.
