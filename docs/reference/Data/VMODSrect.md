##### Data Type : Navigators
##### VMODSrect - Bounding rectangle for Navigator graphical objects.
---
```
#include <vmods.h>
```

**Definition :**

typedef struct {
   LONG left;
   LONG top;
   LONG right;
   LONG bottom;
} VMODSrect;

**Description :**

This structure is used to store the bounding rectangle for a graphical object in a Navigator CD record.
