##### Data Type : Database
##### DBQUOTAINFO - Database quota structure.
---
```
#include <nsfdata.h>
```
**Description :**

This structure is used by NSFDbQuotaSetExt() and returned by NSFDbQuotaGet().  
It contains the database quota information.
