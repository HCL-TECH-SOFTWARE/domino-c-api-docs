##### Data Type : Standard
##### BOOLBYTE - Boolean value occupying one BYTE.
---
```
#include <global.h>
```

**Definition :**
```
#define BOOLBYTE BYTE
```

**Description :**

A BOOLBYTE occupies one BYTE, and may be set to either TRUE or FALSE.
