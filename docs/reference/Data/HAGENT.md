##### Data Type : Agents
##### HAGENT - Handle to an open Agent.
---
```
#include <agents.h>
```
**Description :**

An Agent handle is used to refer to an open agent.  The Agent handle is created 
by AgentOpen(), and must be released by a call to AgentClose().
