##### Data Type : Composite Data
##### CDPOINT - Coordinates defining a polygon within an AREA element.
---
```
#include <editods.h>
```
**Description :**

This structure defines a polygon within a CDAREAELEMENT if the AREA_SHAPE_xxx 
is an AREA_SHAPE_POLYGON.
