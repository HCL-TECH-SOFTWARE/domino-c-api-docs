##### Data Type : Handles
##### DBHANDLE - Database handle.
---
```
#include <nsfdata.h>
```

**Definition :**
```
#define DBHANDLE HANDLE
```

**Description :**

A DBHANDLE is the method by which an API program refers to a Domino database, once the database is opened.
