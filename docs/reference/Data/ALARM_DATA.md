##### Data Type : Alarms
##### ALARM_DATA - Alarm structure.
---
```
#include <alarm.h>
```
**Description :**

Message structure sent TO THE CLIENT, one per alarm.  There is an array of 
these structures pointed to by ALARM_MSG.
